# パーソナライゼーションインサイトレポートの使用

Automated Personalizationおよび自動ターゲットアクティビティでは、高度な機械学習を使用して、個々の顧客プロファイルと類似のプロファイルを持つ以前の訪問者の行動に基づいて、各訪問者に最適なエクスペリエンスを提供します。 パーソナライゼーションインサイトレポートでは、これらのモデルがどのように決定を下すかに関する情報を提供します。
