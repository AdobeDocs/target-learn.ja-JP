# パーソナライゼーションインサイトレポートの使用

Automated Personalizationと自動ターゲットのアクティビティでは、高度な機械学習を使用して、個々の顧客プロファイルと、類似したプロファイルを持つ以前の訪問者の行動に基づいて、各訪問者に最適なエクスペリエンスを提供します。 パーソナライゼーションインサイトレポートでは、これらのモデルがどのように決定を下すかに関する情報を提供します。
