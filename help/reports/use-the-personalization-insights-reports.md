# 個人用設定インサイトレポートの使用

Automated Personalizationと自動ターゲットのアクティビティは、高度な機械学習機能を使用して、個々の顧客のプロファイルと類似のプロファイルを持つ以前の訪問者の行動に基づき、各訪問者に最も合わせたエクスペリエンスを提供します。 パーソナライゼーションインサイトレポートは、これらのモデルが決定を行う方法に関する情報を提供します。
