# [!DNL Adobe Target] でのプロパティの設定

このビデオでは、次の方法を学びます。

* [!DNL Adobe Target] でのプロパティの作成

## 対象読者

* Administrator

## 要件

ユーザーがプロパティを作成するには、次の権限が必要です。

* 製品管理者（製品プロファイル管理者と混同しないこと）
* システム管理者
