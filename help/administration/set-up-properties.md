# Adobe Targetでのプロパティの設定

## 説明

このビデオでは、次の方法を学びます。

* Adobe Targetでのプロパティの作成

## 対象オーディエンス

* 管理者

## 要件

ユーザーがプロパティを作成するには、次の権限が必要です。

* 製品管理者（製品プロファイル管理者と混同しないこと）
* システム管理者
