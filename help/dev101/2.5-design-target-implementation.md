# ターゲット導入の設計

## 説明

このビデオでは、次の方法を学習します。

* 標準サイト（複数ページ）と単一ページアプリ(SPA)の環境での実装の違いを特定
* プロセスを適用して、人物ベースのキャンペーン(mbox3rdPartyID)を実装します。
* システムアーキテクチャ/設計ドキュメントの関連情報の特定
* 導入に関するレポート戦略の分析
